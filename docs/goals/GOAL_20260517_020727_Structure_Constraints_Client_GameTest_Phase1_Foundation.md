## Objective
Build the first client GameTest foundation for actually validating Terrasect's structure constraints.

## Phase 1 scope
Keep this phase intentionally simple and deterministic:

1. Use a static seed so the test worlds are repeatable.
2. Establish a vanilla ground-truth run that can be used to locate structures generated by default.
3. Teleport to a few of those vanilla structures and capture screenshots as test artifacts.
4. Add a constrained/high-density scenario that increases the amount of structures generated.
5. Teleport to a few of the high-density structures and capture screenshots as test artifacts.
6. Keep the entire first phase focused on a reliable screenshot-based feedback loop, not on fancy assertions or full end-to-end coverage yet.
