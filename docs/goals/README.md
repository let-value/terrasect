# Terrasect Goals Index

This folder stores self-contained task contracts for delegated external-agent work,
following the protocol in [`docs/AGENT_WORKFLOW.md`](../AGENT_WORKFLOW.md).

A goal file (`GOAL_[YYYYMMDD]_[HHMMSS]_[ShortDescription].md`) is created per
active task and stays here until completed. Once a topic is done, compact its
outcome into a one-line entry below (or drop it if it left no lasting decisions)
and delete the full file — full discussion/history lives in Git and PR comments.
